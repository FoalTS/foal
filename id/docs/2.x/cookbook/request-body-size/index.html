Request Body Size

By default, FoalTS only accepts request bodies lower than 100kb. This value can be increased by using the configuration key settings.bodyParser.limit. If a number is provided, then the value specifies the number of bytes. If it is a string, the value is passed to the bytes library for parsing.

  • YAML
  • JSON
  • JS
settings:
  bodyParser:
    limit: 50mb
{
  "settings": {
    "bodyParser": {
      "limit": "50mb"
    }
  }
}
module.exports = {
  settings: {
    bodyParser: {
      limit: "50mb"
    }
  }
}
